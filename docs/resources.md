## 学习路线推荐

### 初级阶段（0-2 年）

1. F01 JavaScript + F02 TypeScript
2. F03 Browser + F04 Network
3. E06 React 或 E07 Vue
4. E01 Build Tools + E04 Code Quality

### 中级阶段（2-5 年）

1. E02 Monorepo + E03 CI/CD
2. E05 Design System + E10 Node.js/BFF
3. A05 Data & State
4. A03 Performance + A04 Quality

### 高级阶段（5 年以上）

1. A01 System Architecture + A02 Micro Frontend
2. A06 Observability
3. E09 AI Engineering
4. L01 Business + L02 Team + L03 Strategy
