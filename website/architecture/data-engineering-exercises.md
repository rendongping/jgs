曝光埋点指元素进入用户可视区。

### 第 2 题

::: details 查看答案与解析

:::

### 第 3 题

::: details 查看答案与解析
**答案：B**

北极星指标是最能反映产品核心价值的单一指标。
:::

### 第 4 题

::: details 查看答案与解析
**答案：B**

页面加载时间是护栏指标，防止实验带来性能副作用。

### 第 5 题

::: details 查看答案与解析
**答案：B**

口径不一致会导致数据分析结果不可信。
:::

### 第 6 题

::: details 查看答案与解析
**参考思路**：
- 假设：新颜色提升点击率。
- 分流：随机 50% 用户看到新颜色。
- 指标：点击率（核心）、页面停留时长（护栏）。
- 运行：收集足够样本，计算统计显著性。
- 决策：显著正向则全量，否则保留原方案。
:::

### 第 7 题

::: details 查看答案与解析
**参考思路**：
- 查看漏斗：访问 → 点击 → 下单 → 支付。
- 定位转化率下降的环节。
- 分析该环节的埋点、性能、错误、用户反馈。
- 用下钻分析按渠道、设备、版本拆分。
:::

### 第 8 题

**参考方案**：
- 曝光：商品主图、推荐商品进入可视区。
- 点击：加入购物车、立即购买、收藏、分享。
- 转化：下单成功、支付成功。
- 属性：商品 ID、价格、位置、用户 ID。

### 第 9 题

**参考体系**：
- 业务：MRR、付费转化率、客户留存率。
- 产品：功能使用率、任务完成率、NPS。
- 技术：页面加载时间、接口成功率、错误率。

### 第 10 题

**参考方案**：
- 建立埋点规范文档和事件字典。
- 使用类型化的埋点 SDK，编译时校验。
- 埋点验收流程：开发自测、QA 验收、数据校验。
- 定期审计埋点有效性和口径。

### 第 11 题

::: details 查看答案与解析
**参考思路**：
- 实时数据采集：Kafka + Flink。
- 指标计算：按维度聚合。
- 可视化：ECharts/AntV 实时刷新。
- 下钻：支持时间、渠道、版本等维度筛选。
:::
