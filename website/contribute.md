# 参与贡献

前端架构师知识库欢迎社区贡献。无论是修正错误、补充案例、新增题目，还是改进网站体验，你的参与都能帮助更多学习者。

## 贡献方式

### 1. 提交 Issue

发现内容错误、死链、排版问题，或有新领域/新题目建议？请通过 GitHub Issue 反馈：

- [内容错误反馈](https://github.com/rendongping/jgs/issues/new?template=content-error.yml)
- [新增领域建议](https://github.com/rendongping/jgs/issues/new?template=new-domain.yml)
- [功能建议](https://github.com/rendongping/jgs/issues/new?template=feature.yml)

### 2. 提交 Pull Request

1. Fork 仓库并克隆到本地。
2. 根据 `docs/STYLE-GUIDE.md` 编辑或新增 Markdown 文件。
3. 运行 `bash scripts/lint-docs.sh` 检查文档规范。
4. 提交 PR，并附上修改说明与价值描述。

### 3. 网页快速反馈

在任意学习文档页面底部，点击「改进此页」即可跳转到 GitHub 编辑界面。

## 贡献奖励

- 首次合并贡献者将加入项目贡献者名单。
- 高质量内容贡献可申请在首页「贡献者」区域展示。

## 行为准则

- 保持友善、尊重不同技术观点。
- 引用外部资料时注明出处。
- 避免抄袭，确保内容可合法共享。
